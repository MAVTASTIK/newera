## 什么是 New era？



###         &emsp;New era 是一个为了解决以太坊在商业应用时遇到瓶颈的应用链，New era 100%和以太坊的公链保持兼容，同时为了能加速链上的成交，New era 全新设计了一个新的 Layer-2 的交易解决方案，将以太坊方的上链数据进行交易加速。

<br>
<br>

###       &emsp;欢迎更多愿意提升以太坊生态质量的人加入我们，参与这个对以太坊来说意义重大的项目。
